## 3. Refinamientos Tipográficos

### 3.1 Font Pairing (mantener Plus Jakarta Sans)
- Hero: Weight 800, size más grande (5xl → 6xl en desktop)
- Subheadings: Weight 600, tracking -0.02em
- Body: Weight 400, line-height 1.7, tracking 0.01em
- CTAs: Weight 600, uppercase, letter-spacing 0.05em

### 3.2 Text Shadows
- Reducir text-shadow intensity ( actuales muy fuertes: 0 2px 20px)
- Alternativa: usar gradient text para títulos del hero
- Ejemplo: `background: linear-gradient(180deg, #fff, #88d4e8); -webkit-background-clip: text`

### 3.3 Reading Hierarchy
```
Hero H1: 4xl-6xl, 800 weight, gradient o outline
Section H2: 3xl-4xl, 700 weight, white
Card H3: xl, 600 weight, white/90
Body: base-lg, 400 weight, white/70-80
Caption: sm, 500 weight, white/50-60
```
